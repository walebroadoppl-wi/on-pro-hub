# on-pro-hub

## 项目简介

on-pro-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的页面内容涵盖多种实用场景，每个页面均为独立、自包含的 HTML 文件，便于直接查看或做进一步处理。本仓库不针对特定域名或单一网站，旨在提供一个灵活、干净的页面集合。

## 目录说明

```
on-pro-hub/
├── pages/        # 存放所有 HTML 页面文件
├── assets/       # 页面所需的公共资源（如图片、样式、脚本）
├── docs/         # 相关文档与说明
└── README.md     # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名通常反映页面用途。
- `assets/`：如有页面需要共用资源，则存放于此；部分页面也可能内嵌所有资源。
- `docs/`：包含页面使用方式、更新记录等补充信息。

## 页面归档说明

- 每个页面均为标准 HTML 格式，兼容主流现代浏览器。
- 页面内容可能涉及工具、展示、参考等不同类型，彼此之间无依赖关系。
- 归档的页面会定期更新或新增，但不对其可用性做长期保证。
- 页面中不包含外部跟踪、广告或恶意代码，仅提供纯粹的内容展示。
